# DashboardService
**Size:** S (1h)

## Description
Static service class aggregating dashboard data from Order/OrderItem.

## Files
- `app/Services/DashboardService.php`

## Methods
- `getStats()`: returns `['total_revenue' => float, 'total_orders' => int, 'today_orders' => int, 'pending_orders' => int]`. Revenue = sum of `orders.total` where `payment_status=paid`. Orders count all. Today = whereDate today. Pending = where status=pending.
- `getRevenueChart(int $days = 7)`: returns array of `['date' => 'Y-m-d', 'label' => 'd/m', 'total' => float]` for last N days. Group by date, sum total where payment_status=paid. Fill zero for days with no orders.
- `getTopProducts(int $limit = 5)`: returns Collection from `order_items` joined with products, where order status != cancelled. Group by product_id, sum quantity as total_qty, sum line_total as total_revenue. Order by total_qty desc. Limit 5. Select product_name, total_qty, total_revenue.
- `getRecentOrders(int $limit = 10)`: returns `Order::latest()->limit(10)->get()`.

## Acceptance Criteria
- [ ] All 4 methods return correct data types
- [ ] Revenue only counts paid orders
- [ ] Chart fills zero-days
- [ ] Top products excludes cancelled orders
