## Size: M (2h)

## Description

Full resource CRUD for categories at `/admin/categories`.

## Files

- `app/Http/Controllers/Admin/CategoryController.php` — index, create, store, edit, update, destroy
- `resources/views/admin/categories/index.blade.php` — table listing with parent name, status badge, edit/delete buttons
- `resources/views/admin/categories/create.blade.php` — form: name[en], name[vi], slug, parent_id select (only top-level categories), image upload, sort_order, status select
- `resources/views/admin/categories/edit.blade.php` — same form, pre-filled

## Acceptance criteria

- [ ] CRUD all works: create with parent, edit, list shows hierarchy, delete blocks if has products
- [ ] Slug auto-generates from Vietnamese name if empty
- [ ] Image upload to `storage/app/public/categories/`
- [ ] Validation: name required, slug unique, parent must be top-level (no 3-level nesting)
- [ ] All strings via `__()`

## Definition of Done

Code implemented, views compile, CRUD functional
