## Routes to add

### Admin (inside existing `admin` group with `auth` middleware)

- `Route::resource('categories', Admin\CategoryController::class)` named `admin.categories.*`
- `Route::resource('attributes', Admin\AttributeController::class)` named `admin.attributes.*`
- `Route::resource('products', Admin\ProductController::class)` named `admin.products.*`
- `POST admin/products/{product}/variants/generate` named `admin.products.variants.generate`
- `POST admin/products/{product}/images` named `admin.products.images.upload`
- `POST admin/products/{product}/images/reorder` named `admin.products.images.reorder`
- `POST admin/products/{product}/images/{image}/primary` named `admin.products.images.primary`
- `DELETE admin/products/{product}/images/{image}` named `admin.products.images.destroy`

### Public (inside existing `{locale}` group)

- `GET /{locale}/products` → `ProductController@index` named `products.index`
- `GET /{locale}/category/{slug}` → `ProductController@byCategory` named `products.category`
- `GET /{locale}/product/{slug}` → `ProductController@show` named `products.show`

### API

- `GET /api/product/{product}/variant-price` → `Api\VariantPriceController` named `api.variant-price`
