## Description

Build the full block editor admin interface: a controller with edit, update, and image upload endpoints, and a standalone Blade view powered by Alpine.js and Sortable.js. The editor allows admins to visually compose a page by adding, removing, reordering, and configuring blocks with a drag-and-drop interface.

## Acceptance Criteria

- [ ] `BlockEditorController` has `edit`, `update`, and `uploadImage` methods
- [ ] Editor view loads with existing blocks pre-populated from the site
- [ ] Block palette sidebar shows all 11 block types with icons and labels
- [ ] Clicking a block type adds it to the page with default content
- [ ] Blocks can be reordered via drag-and-drop using Sortable.js
- [ ] Each block can be expanded to show its edit form
- [ ] Form fields are auto-generated from block type field definitions in config
- [ ] All 7 field types render correctly: text_i18n, textarea_i18n, text, number, image, select, repeater
- [ ] Image upload works via AJAX and returns the stored URL
- [ ] Save button sends blocks JSON to the server and shows success/error feedback
- [ ] Blocks can be deleted with confirmation
- [ ] UI is clean and professional using Tailwind CSS

## Technical Details

### BlockEditorController

File: `app/Http/Controllers/Admin/BlockEditorController.php`

```php
namespace App\Http\Controllers\Admin;

use App\Http\Controllers\Controller;
use App\Models\Site;
use Illuminate\Http\Request;

class BlockEditorController extends Controller
{
    public function edit(Site $site)
    {
        $blocks = $site->blocks ?? [];
        $blockTypes = config('blocks');

        return view('admin.sites.editor', [
            'site' => $site,
            'blocksJson' => json_encode($blocks),
            'blockTypes' => $blockTypes,
            'blockTypesJson' => json_encode($blockTypes),
        ]);
    }

    public function update(Request $request, Site $site)
    {
        $request->validate([
            'blocks' => 'required|json',
        ]);

        $site->update([
            'blocks' => json_decode($request->input('blocks'), true),
        ]);

        return response()->json(['success' => true]);
    }

    public function uploadImage(Request $request, Site $site)
    {
        $request->validate([
            'image' => 'required|image|max:5120', // 5MB max
        ]);

        $path = $request->file('image')->store("sites/{$site->id}", 'public');

        return response()->json([
            'url' => asset('storage/' . $path),
        ]);
    }
}
```

#### Alpine Component: `blockEditor()`

```javascript
function blockEditor() {
    return {
        blocks: JSON.parse('{{ $blocksJson }}'),
        blockTypes: JSON.parse('{{ $blockTypesJson }}'),
        activeBlockId: null,
        saving: false,
        message: '',

        init() {
            this.$nextTick(() => {
                Sortable.create(document.getElementById('blocks-list'), {
                    handle: '.drag-handle',
                    animation: 150,
                    onEnd: (evt) => {
                        const moved = this.blocks.splice(evt.oldIndex, 1)[0];
                        this.blocks.splice(evt.newIndex, 0, moved);
                        this.blocks.forEach((b, i) => b.order = i);
                    }
                });
            });
        },

        addBlock(type) {
            const blockType = this.blockTypes[type];
            this.blocks.push({
                id: 'block_' + Math.random().toString(36).substr(2, 9),
                type: type,
                order: this.blocks.length,
                content: JSON.parse(JSON.stringify(blockType.default_content))
            });
        },

        removeBlock(id) {
            if (!confirm('Are you sure?')) return;
            this.blocks = this.blocks.filter(b => b.id !== id);
            this.blocks.forEach((b, i) => b.order = i);
        },

        toggleBlock(id) {
            this.activeBlockId = this.activeBlockId === id ? null : id;
        },

        updateBlockContent(id, field, value) {
            const block = this.blocks.find(b => b.id === id);
            if (block) {
                block.content[field] = value;
            }
        },

        async save() {
            this.saving = true;
            this.message = '';
            try {
                const res = await fetch(`/admin/sites/{{ $site->id }}/blocks`, {
                    method: 'PUT',
                    headers: {
                        'Content-Type': 'application/json',
                        'X-CSRF-TOKEN': document.querySelector('meta[name="csrf-token"]').content,
                    },
                    body: JSON.stringify({ blocks: JSON.stringify(this.blocks) }),
                });
                const data = await res.json();
                this.message = data.success ? 'Blocks saved!' : 'Error saving.';
            } catch (e) {
                this.message = 'Error saving blocks.';
            }
            this.saving = false;
        },

        async uploadImage(blockId, field, event) {
            const file = event.target.files[0];
            if (!file) return;
            const formData = new FormData();
            formData.append('image', file);
            try {
                const res = await fetch(`/admin/sites/{{ $site->id }}/blocks/upload`, {
                    method: 'POST',
                    headers: {
                        'X-CSRF-TOKEN': document.querySelector('meta[name="csrf-token"]').content,
                    },
                    body: formData,
                });
                const data = await res.json();
                this.updateBlockContent(blockId, field, data.url);
            } catch (e) {
                alert('Upload failed');
            }
        },

        getPreview(block) {
            // Return first text field value, truncated
            const content = block.content || {};
            for (const key of Object.keys(content)) {
                const val = content[key];
                if (typeof val === 'string' && val.length > 0) {
                    return val.substring(0, 60) + (val.length > 60 ? '...' : '');
                }
                if (typeof val === 'object' && val !== null && !Array.isArray(val)) {
                    const locVal = val['vi'] || val['en'] || '';
                    if (locVal) return locVal.substring(0, 60) + (locVal.length > 60 ? '...' : '');
                }
            }
            return '(empty)';
        }
    };
}
```

#### Field Type Rendering

The expanded block form auto-generates inputs based on `blockTypes[type].fields`:

- **`text_i18n`**: Two side-by-side inputs labeled "VI" and "EN"
  ```html
  <div class="grid grid-cols-2 gap-2">
      <div><label>VI</label><input x-model="block.content[field.name].vi"></div>
      <div><label>EN</label><input x-model="block.content[field.name].en"></div>
  </div>
  ```

- **`textarea_i18n`**: Two side-by-side textareas labeled "VI" and "EN"

- **`text`**: Single text input
  ```html
  <input type="text" x-model="block.content[field.name]">
  ```

- **`number`**: Number input
  ```html
  <input type="number" x-model="block.content[field.name]">
  ```

- **`image`**: File input with preview thumbnail
  ```html
  <div>
      <img x-show="block.content[field.name]" :src="block.content[field.name]" class="h-20 mb-2 rounded">
      <input type="file" accept="image/*" @change="uploadImage(block.id, field.name, $event)">
  </div>
  ```

- **`select`**: Dropdown select (e.g., for category_id in products block)
  ```html
  <select x-model="block.content[field.name]">
      <option value="">-- Select --</option>
      <!-- Options populated from field.options or fetched -->
  </select>
  ```

- **`repeater`**: Dynamic list with add/remove buttons, each row contains sub-fields
  ```html
  <div>
      <template x-for="(item, index) in block.content[field.name]" :key="index">
          <div class="flex gap-2 items-start mb-2 p-2 border rounded">
              <!-- render sub-fields for each item -->
              <button @click="block.content[field.name].splice(index, 1)">✕</button>
          </div>
      </template>
      <button @click="block.content[field.name].push({})">+ Add item</button>
  </div>
  ```
