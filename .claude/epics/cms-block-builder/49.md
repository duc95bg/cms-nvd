#### Test 4: `test_block_editor_saves_blocks_json`

- Authenticate as admin
- Create a site
- PUT `/admin/sites/{site}/blocks` with JSON blocks payload
- Assert JSON response `{success: true}`
- Assert site's blocks column updated in DB

```php
public function test_block_editor_saves_blocks_json()
{
    $user = User::factory()->create(/* admin */);
    $site = Site::factory()->create();

    $blocks = [
        ['id' => 'b1', 'type' => 'hero', 'order' => 0, 'content' => ['title' => ['vi' => 'Test', 'en' => 'Test']]],
    ];

    $response = $this->actingAs($user)->putJson(route('admin.sites.blocks.update', $site), [
        'blocks' => json_encode($blocks),
    ]);

    $response->assertJson(['success' => true]);
    $this->assertDatabaseHas('sites', ['id' => $site->id]);
    $site->refresh();
    $this->assertCount(1, $site->blocks);
}
```
