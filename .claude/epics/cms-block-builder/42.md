## Description

Create 11 Blade partial views in `resources/views/blocks/`, one per block type defined in `config/blocks.php`. Each partial receives a `$block` variable containing the block's content array and renders responsive HTML using Tailwind CSS classes. All i18n text fields are resolved using `app()->getLocale()`.

## Acceptance Criteria

- [ ] 11 Blade files exist at `resources/views/blocks/{type}.blade.php`
- [ ] Each partial renders valid HTML when passed a well-formed `$block` array
- [ ] All i18n fields resolve to the current app locale with fallback
- [ ] All partials use Tailwind CSS classes for styling
- [ ] Responsive design: partials look correct on mobile and desktop
- [ ] `products.blade.php` queries the database for active products
- [ ] `html.blade.php` renders raw HTML safely with `{!! !!}`
- [ ] `spacer.blade.php` respects configurable height

## Technical Details

### Directory

`resources/views/blocks/`

### 1. `hero.blade.php`

Full-width section with background image, overlay, centered text content.

```blade
<section class="relative bg-cover bg-center min-h-[400px] flex items-center"
    style="background-image: url('{{ $block['background_image'] ?? '' }}')">
    <div class="absolute inset-0 bg-black/40"></div>
    <div class="relative container mx-auto px-4 text-center text-white">
        <h1 class="text-4xl md:text-5xl font-bold mb-4">
            {{ data_get($block, 'title.' . app()->getLocale(), '') }}
        </h1>
        <p class="text-xl mb-6">
            {{ data_get($block, 'subtitle.' . app()->getLocale(), '') }}
        </p>
        @if(data_get($block, 'button_url'))
            <a href="{{ $block['button_url'] }}"
               class="inline-block bg-white text-gray-900 px-8 py-3 rounded-lg font-semibold hover:bg-gray-100 transition">
                {{ data_get($block, 'button_label.' . app()->getLocale(), '') }}
            </a>
        @endif
    </div>
</section>
```

### 4. `gallery.blade.php`

Responsive grid of images from `$block['images']` array.

```blade

            {{ data_get($block, 'heading.' . app()->getLocale(), '') }}
        </h2>
    @endif
    <div class="max-w-6xl mx-auto grid grid-cols-2 md:grid-cols-3 gap-4">
        @foreach(($block['images'] ?? []) as $image)
            <div class="overflow-hidden rounded-lg">
                <img src="{{ $image['url'] ?? '' }}"
                     alt="{{ data_get($image, 'alt.' . app()->getLocale(), '') }}"
                     class="w-full h-48 object-cover hover:scale-105 transition duration-300">
            </div>
        @endforeach
    </div>
</section>
```

### 6. `products.blade.php`

Queries active products by category and renders product cards.

```blade
@php
    $products = \App\Models\Product::active()
        ->when(data_get($block, 'category_id'), fn($q, $catId) => $q->where('category_id', $catId))
        ->limit($block['count'] ?? 4)
        ->get();
@endphp
<section class="py-12 px-4">
    <div class="max-w-6xl mx-auto">
        <h2 class="text-3xl font-bold text-center mb-8">
            {{ data_get($block, 'heading.' . app()->getLocale(), '') }}
        </h2>
        <div class="grid grid-cols-2 md:grid-cols-4 gap-6">
            @foreach($products as $product)
                @include('products._card', ['product' => $product])
            @endforeach
        </div>
    </div>
</section>
```
