## Description

Create the admin settings management page: a controller that loads and saves global site settings using `SettingService`, and a Blade form view with sections for branding, contact info, and social media links. Includes file upload handling for logo and favicon.

## Acceptance Criteria

- [ ] `SettingsController` has `edit` and `update` methods
- [ ] Settings page loads all current settings from the database
- [ ] Form has 3 sections: Branding, Contact, Social
- [ ] Logo and favicon can be uploaded with preview thumbnails
- [ ] i18n fields (site_name, tagline, address) have VI and EN inputs
- [ ] Saving updates all settings via `SettingService::set()`
- [ ] Success message is shown after saving
- [ ] File uploads are stored in `public` disk under `settings/` directory
- [ ] Validation prevents invalid data (e.g., URL format for social links)

## Technical Details

### SettingsController

File: `app/Http/Controllers/Admin/SettingsController.php`

```php
namespace App\Http\Controllers\Admin;

use App\Http\Controllers\Controller;
use App\Services\SettingService;
use Illuminate\Http\Request;

class SettingsController extends Controller
{
    public function edit()
    {
        $settings = SettingService::all();

        return view('admin.settings.edit', [
            'settings' => $settings,
        ]);
    }

    public function update(Request $request)
    {
        $request->validate([
            'site_name' => 'required|array',
            'site_name.vi' => 'required|string|max:255',
            'site_name.en' => 'required|string|max:255',
            'tagline' => 'nullable|array',
            'tagline.vi' => 'nullable|string|max:255',
            'tagline.en' => 'nullable|string|max:255',
            'email' => 'nullable|email|max:255',
            'phone' => 'nullable|string|max:50',
            'address' => 'nullable|array',
            'address.vi' => 'nullable|string|max:500',
            'address.en' => 'nullable|string|max:500',
            'facebook' => 'nullable|url|max:500',
            'instagram' => 'nullable|url|max:500',
            'youtube' => 'nullable|url|max:500',
            'tiktok' => 'nullable|url|max:500',
            'logo' => 'nullable|image|max:2048',
            'favicon' => 'nullable|image|max:1024',
        ]);

        // Handle file uploads
        if ($request->hasFile('logo')) {
            $logoPath = $request->file('logo')->store('settings', 'public');
            SettingService::set('logo', asset('storage/' . $logoPath));
        }

        if ($request->hasFile('favicon')) {
            $faviconPath = $request->file('favicon')->store('settings', 'public');
            SettingService::set('favicon', asset('storage/' . $faviconPath));
        }

        // Save text settings
        $textSettings = ['site_name', 'tagline', 'email', 'phone', 'address', 'facebook', 'instagram', 'youtube', 'tiktok'];

        foreach ($textSettings as $key) {
            if ($request->has($key)) {
                SettingService::set($key, $request->input($key));
            }
        }

        return redirect()->route('admin.settings.edit')
            ->with('success', __('Settings saved'));
    }
}
```
