{

### SettingService

File: `app/Services/SettingService.php`

```php
class SettingService
{
    protected static array $cache = [];

    public static function get(string $key, mixed $default = null): mixed
    {
        if (isset(static::$cache[$key])) {
            return static::$cache[$key];
        }
        $setting = Setting::where('key', $key)->first();
        $value = $setting ? $setting->value : $default;
        static::$cache[$key] = $value;
        return $value;
    }

    public static function set(string $key, mixed $value): void
    {
        Setting::updateOrCreate(['key' => $key], ['value' => $value]);
        static::$cache[$key] = $value;
    }

    public static function all(): array
    {
        $settings = Setting::all()->pluck('value', 'key')->toArray();
        static::$cache = array_merge(static::$cache, $settings);
        return $settings;
    }
}
```

### config/blocks.php

File: `config/blocks.php`

Return an associative array with 11 block type keys: `hero`, `text`, `image`, `gallery`, `features`, `products`, `about`, `contact`, `cta`, `spacer`, `html`.

Each entry structure:
```php
'hero' => [
    'label' => ['vi' => 'Banner chính', 'en' => 'Hero Banner'],
    'icon' => '🖼️',
    'default_content' => [
        'title' => ['vi' => '', 'en' => ''],
        'subtitle' => ['vi' => '', 'en' => ''],
        'background_image' => '',
        'button_label' => ['vi' => '', 'en' => ''],
        'button_url' => '',
    ],
    'fields' => [
        ['name' => 'title', 'type' => 'text_i18n'],
        ['name' => 'subtitle', 'type' => 'text_i18n'],
        ['name' => 'background_image', 'type' => 'image'],
        ['name' => 'button_label', 'type' => 'text_i18n'],
        ['name' => 'button_url', 'type' => 'text'],
    ],
],
```

Field types used across all blocks: `text_i18n`, `textarea_i18n`, `text`, `number`, `image`, `select`, `repeater`.
