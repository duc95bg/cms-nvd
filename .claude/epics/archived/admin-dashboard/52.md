# DashboardController + view
**Size:** M (2h)

## Description
Controller calls DashboardService, passes data to a rich admin view.

## Files
- `app/Http/Controllers/Admin/DashboardController.php`
- `resources/views/admin/dashboard.blade.php`

## Controller
```php
class DashboardController extends Controller {
    public function index(): View {
        return view('admin.dashboard', [
            'stats' => DashboardService::getStats(),
            'chart' => DashboardService::getRevenueChart(),
            'topProducts' => DashboardService::getTopProducts(),
            'recentOrders' => DashboardService::getRecentOrders(),
        ]);
    }
}
```

## View layout
Standalone admin Blade + Tailwind CDN. Include `@include('admin.partials.nav')`.

**Stat cards (4-col grid):**
- Total Revenue: 💰 icon, formatted price, bg-green-50
- Total Orders: 📦 icon, count, bg-blue-50
- Today's Orders: 📅 icon, count, bg-yellow-50
- Pending: ⏳ icon, count, bg-red-50

**Revenue chart (below stats):**
- Heading "Doanh thu 7 ngày"
- Horizontal bar chart: each day = row with label (d/m) + bar (div bg-blue-500, width relative to max) + amount
- Max day = 100% width, others proportional

**2-column grid below:**
- Left: Top 5 products table (rank, name, qty sold, revenue)
- Right: Recent 10 orders table (order#→link, customer, total, status badge, date)
