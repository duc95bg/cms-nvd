## Size: L (4h)

## Description

The most complex admin view — full product management including attribute selection, variant generation, image upload/reorder, and inline variant editing.

## Files

- `app/Http/Controllers/Admin/ProductController.php` — index, create, store, edit, update, destroy, generateVariants, uploadImage, deleteImage, reorderImages
- `resources/views/admin/products/index.blade.php` — table listing with category name, price, stock summary, status
- `resources/views/admin/products/create.blade.php` — form: name[en/vi], slug, short_description[en/vi], description[en/vi], category_id, base_price, status, featured, sort_order, attribute multi-select
- `resources/views/admin/products/edit.blade.php` — same as create PLUS: variant table (SKU/price/stock/status per row), image gallery with upload/reorder/set-primary/delete

## Key details

- **Variant generation:** `POST admin/products/{product}/variants/generate` — compute cartesian product of selected attribute values, create variants that don't exist yet (match by attribute value set), don't delete existing variants.
- **Variant table:** rendered as HTML table rows. Each row: SKU (input), price (input, placeholder shows base_price), stock (input), status (select), image (file input). Save all variants in one form submit along with product data.
- **Image management:** upload via `POST admin/products/{product}/images`, reorder via `POST admin/products/{product}/images/reorder` (receives sorted IDs), delete via `DELETE admin/products/{product}/images/{image}`. Max 10 images, max 5MB each. Set primary via `POST admin/products/{product}/images/{image}/primary`.
- **Slug** auto-generate from vi name if blank.

## Acceptance criteria

- [ ] Create product with all fields, assign category
- [ ] Select attributes → generate variants creates correct count (cartesian)
- [ ] Edit variant SKU/price/stock inline in table
- [ ] Upload images, set primary, reorder, delete
- [ ] Validation: name required, slug unique, base_price >= 0, at least 1 category
- [ ] All strings `__()`
