## Size: M (2h)

## Description

Public-facing product listing pages.

## Files

- `app/Http/Controllers/ProductController.php` (public) — index (all products), byCategory (filtered)
- `resources/views/products/index.blade.php` — grid layout with product cards, sidebar category tree, sort dropdown, pagination
- `resources/views/products/_card.blade.php` — shared partial: image, name, price range ("từ X₫"), stock badge

## Key details

- Extend `templates.layouts.base` (already exists from cms-landing-builder)
- Only show products with `status=active`
- Price display: if variants exist, show "từ {min}₫" / "from ${min}"; if no variants, show base_price
- Stock badge: "Còn hàng"/"In stock" if any active variant has stock > 0, else "Hết hàng"/"Out of stock"
- Category sidebar: 2-level tree, current category highlighted
- Pagination: 12/page, Tailwind default pagination view
- Sort: newest, price asc, price desc, name
- Breadcrumb on byCategory page

## Acceptance criteria

- [ ] `/{locale}/products` shows paginated grid of active products
- [ ] `/{locale}/category/{slug}` filters by category
- [ ] Price range displays correctly
- [ ] Stock badge shows correctly
- [ ] Category sidebar renders 2-level tree
- [ ] Sort works (4 options)
- [ ] Responsive: 4-col desktop, 2-col mobile

## Definition of Done

Code implemented, views compile, listing renders correctly with seeded data
