# Task: Data layer — migrations and Eloquent models

## Description
Create the database schema and Eloquent models that the rest of the epic builds on. The content column on `sites` is a JSON blob; no per-locale columns. `Site::t()` is the single localization entry point used by every template Blade.

## Acceptance Criteria
- [ ] `templates` table migration exists: `id, name, type, view, default_content JSON, timestamps`
- [ ] `sites` table migration exists: `id, user_id FK → users cascadeOnDelete, template_id FK → templates restrictOnDelete, slug UNIQUE, content JSON, published BOOL default false, timestamps`
- [ ] `media` table migration exists: `id, site_id FK cascadeOnDelete, path, disk default 'public', mime nullable, size unsigned int nullable, timestamps`
- [ ] `Template` model: `$fillable = ['name','type','view','default_content']`, cast `default_content => array`, `sites()` hasMany
- [ ] `Site` model: `$fillable = ['user_id','template_id','slug','content','published']`, casts `content => array`, `published => boolean`; `user()`, `template()`, `media()` relationships
- [ ] `Site::t(string $key, ?string $locale = null, string $default = ''): string` — returns `data_get($content, "$key.$locale")` with fallback to `config('app.fallback_locale')` then `$default`
- [ ] `Media` model: `$table = 'media'`, belongsTo site, `getUrlAttribute()` returns `asset('storage/'.$this->path)`
- [ ] `php artisan migrate` runs cleanly on a fresh DB

## Technical Details
- Files to create:
  - `database/migrations/<ts>_create_templates_table.php`
  - `database/migrations/<ts>_create_sites_table.php`
  - `database/migrations/<ts>_create_media_table.php`
  - `app/Models/Template.php`
  - `app/Models/Site.php`
  - `app/Models/Media.php`
- `Site::t()` contract (keep exact):
  ```php
  public function t(string $key, ?string $locale = null, string $default = ''): string
  {
      $locale = $locale ?: app()->getLocale();
      $fallback = config('app.fallback_locale');
      return (string) (
          data_get($this->content, $key.'.'.$locale)
          ?? data_get($this->content, $key.'.'.$fallback)
          ?? $default
      );
  }
  ```
- Migration timestamps must be > the existing `multi-language` ones so they run last.
