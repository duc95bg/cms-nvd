# 32 — Checkout flow: COD and bank transfer

**Size:** M (2.5h)

## Description

Build the checkout flow including the checkout page with customer form and payment method selection, order creation, and result pages for COD (success page) and bank transfer (bank info page). This task also creates the CheckoutController and the initial OrderController with success/bankTransfer actions. VNPay and PayPal redirects are stubbed here and completed in tasks 33 and 34.

This task depends on task 31 (cart UI must exist for the flow to make sense).

## Acceptance Criteria

1. Checkout page shows a customer information form and payment method selection.
2. Checkout page shows an order summary with items from the cart.
3. Checkout page redirects to cart if the cart is empty.
4. Submitting with COD creates an order and shows the success page.
5. Submitting with bank transfer creates an order and shows bank transfer info page.
6. Submitting with VNPay or PayPal creates an order and redirects to the payment gateway URL (implemented in tasks 33/34).
7. Success page displays order number, status, and items summary.
8. Bank transfer page shows bank details from config and the order number as transfer reference.
9. Customer form validates all required fields.

## Technical Details

### `app/Http/Controllers/CheckoutController.php`

```php
class CheckoutController extends Controller
{
    public function __construct(
        private CartService $cart,
        private OrderService $orderService
    ) {}

    /**
     * GET /checkout
     * Shows the checkout form. Redirects to cart if cart is empty.
     */
    public function index(): View|RedirectResponse
    {
        if ($this->cart->isEmpty()) {
            return redirect()->route('cart.index')
                ->with('error', __('Your cart is empty'));
        }

        return view('checkout.index', [
            'items' => $this->cart->items(),
            'total' => $this->cart->total(),
        ]);
    }

    /**
     * POST /checkout
     * Validates customer data and payment method, creates order,
     * redirects based on payment method.
     */
    public function process(Request $request): RedirectResponse
    {
        $validated = $request->validate([
            'customer_name'    => 'required|string|max:255',
            'customer_email'   => 'required|email|max:255',
            'customer_phone'   => 'required|string|max:20',
            'customer_address' => 'required|string|max:1000',
            'customer_notes'   => 'nullable|string|max:1000',
            'payment_method'   => 'required|in:cod,bank_transfer,vnpay,paypal',
        ]);

        $customer = [
            'name'    => $validated['customer_name'],
            'email'   => $validated['customer_email'],
            'phone'   => $validated['customer_phone'],
            'address' => $validated['customer_address'],
            'notes'   => $validated['customer_notes'] ?? null,
        ];

        try {
            $order = $this->orderService->createOrder(
                $customer,
                $validated['payment_method'],
                $this->cart
            );
        } catch (InsufficientStockException $e) {
            return redirect()->route('cart.index')
                ->with('error', $e->getMessage());
        }

        return match ($validated['payment_method']) {
            'cod'           => redirect()->route('orders.success', $order),
            'bank_transfer' => redirect()->route('orders.bank-transfer', $order),
            'vnpay'         => redirect(PaymentService::createVnpayUrl($order)),
            'paypal'        => redirect(PaymentService::createPaypalOrder($order)),
        };
    }
}
```

### `app/Http/Controllers/OrderController.php` (partial — history/detail added in task 36)

```php
class OrderController extends Controller
{
    /**
     * GET /orders/{order}/success
     * Shows the order success/thank-you page.
     */
    public function success(Order $order): View
    {
        $order->load('items');
        return view('orders.success', compact('order'));
    }

    /**
     * GET /orders/{order}/bank-transfer
     * Shows bank transfer information for the order.
     */
    public function bankTransfer(Order $order): View
    {
        $order->load('items');
        return view('orders.bank-transfer', [
            'order'       => $order,
            'bankName'    => config('payment.bank_name'),
            'bankAccount' => config('payment.bank_account'),
            'bankHolder'  => config('payment.bank_holder'),
        ]);
    }
}
```
