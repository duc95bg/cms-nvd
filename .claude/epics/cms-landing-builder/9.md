# Task: Template seeder with product default_content

## Description
Seed the `templates` table with a realistic `product` template that every developer gets on `db:seed`. The `default_content` JSON is the source of truth for what fields the product template exposes ‚Äî copied into every new site as its starting content.

## Acceptance Criteria
- [ ] `database/seeders/TemplateSeeder.php` exists and is registered in `DatabaseSeeder`
- [ ] Seeder inserts exactly one row for the `product` template on a fresh DB
- [ ] `templates.view = 'templates.product'`
- [ ] `default_content` contains these keys, all translatable leaves keyed `{en, vi}`:
  - `brand.name`
  - `seo.title`, `seo.description`
  - `hero.title`, `hero.subtitle`, `hero.cta_label`
  - `hero.cta_url` (plain string, not translatable)
  - `features.heading`
  - `features.items` ‚Äî indexed list of 3 objects, each with `{ title: {en, vi}, body: {en, vi} }`
  - `pricing.heading`, `pricing.subheading`, `pricing.cta_label`, `pricing.cta_url`
- [ ] Seeder is idempotent: running `db:seed` twice does not duplicate the row (use `Template::updateOrCreate` keyed on `type='product'`)
- [ ] `php artisan db:seed --class=TemplateSeeder` runs cleanly

## Technical Details
- File: `database/seeders/TemplateSeeder.php`
- Update: `database/seeders/DatabaseSeeder.php` ‚Äî call `$this->call(TemplateSeeder::class)`
- Content should use realistic Vietnamese + English copy, not lorem ipsum ‚Äî the seed is what a new site looks like before editing, and weak copy makes the product template look broken.
