# 38 — Lang keys for cart, checkout, and orders

**Size:** XS (0.5h)

## Description

Add all translation keys used by the cart, checkout, and order views to both `lang/en.json` and `lang/vi.json`. This ensures the UI is fully translatable between English and Vietnamese.

This task depends on task 30 (to know the domain terminology) but can be worked on in parallel with most other tasks since it only touches lang files.

## Acceptance Criteria

1. All `__()` keys used in cart, checkout, and order views exist in `lang/en.json`.
2. All `__()` keys used in cart, checkout, and order views exist in `lang/vi.json` with proper Vietnamese translations.
3. No missing translation warnings when browsing the cart/checkout/order pages.
4. Existing translation keys are not removed or modified.
