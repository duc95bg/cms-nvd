# 34 — PayPal payment integration

**Size:** M (2h)

## Description

Implement the PayPal payment gateway integration: the return URL handler (after user approves payment on PayPal), the cancel handler, and CSRF exclusion for PayPal callback routes. This task adds PayPal-specific methods to the PaymentController and ensures the checkout flow correctly redirects to PayPal when that payment method is selected.

This task depends on task 30 (PaymentService must exist). It can be worked on in parallel with tasks 31, 33, 35, 36.

## Acceptance Criteria

1. When a user selects PayPal at checkout, they are redirected to PayPal's approval page.
2. After approving payment on PayPal, the user is redirected back and the payment is captured.
3. On successful capture: order payment_status is updated to "paid", user sees success page.
4. On cancelled payment: user is redirected back with an error message.
5. USD conversion uses the configured exchange rate from `config('payment.paypal.usd_rate')`.
6. PayPal callback routes are excluded from CSRF verification.
7. Payment data (PayPal response) is stored in the order's `payment_data` JSON column.

## Technical Details

### PaymentController additions

Add these methods to `app/Http/Controllers/PaymentController.php` (created in task 33, or create here if working in parallel):

```php
/**
 * GET /payment/paypal/return?token={token}&PayerID={payerId}
 * Handles the return from PayPal after user approves the payment.
 * Captures the payment and updates the order.
 */
public function paypalReturn(Request $request): RedirectResponse
{
    $token = $request->query('token');

    if (!$token) {
        return redirect()->route('home')
            ->with('error', __('Payment verification failed'));
    }

    try {
        $captureData = PaymentService::capturePaypalPayment($token);
    } catch (\Exception $e) {
        Log::error('PayPal capture failed', ['error' => $e->getMessage()]);
        return redirect()->route('cart.index')
            ->with('error', __('Payment failed. Please try again.'));
    }

    // Find order by the PayPal order ID stored during creation
    // The order_number was passed as custom_id or reference_id in the PayPal order
    $referenceId = $captureData['purchase_units'][0]['reference_id'] ?? null;
    $order = Order::where('order_number', $referenceId)->firstOrFail();

    if (($captureData['status'] ?? '') === 'COMPLETED') {
        $order->update([
            'payment_status' => 'paid',
            'payment_data'   => $captureData,
        ]);
        return redirect()->route('orders.success', $order);
    } else {
        $order->update([
            'payment_status' => 'failed',
            'payment_data'   => $captureData,
        ]);
        return redirect()->route('cart.index')
            ->with('error', __('Payment failed. Please try again.'));
    }
}

/**
 * GET /payment/paypal/cancel
 * Handles when the user cancels payment on PayPal.
 */
public function paypalCancel(Request $request): RedirectResponse
{
    return redirect()->route('cart.index')
        ->with('error', __('Payment was cancelled.'));
}
```

### PaymentService::createPaypalOrder() details

Ensure the PayPal order creation in PaymentService (task 30) passes `reference_id` as the order_number so we can look it up on return:

```php
public static function createPaypalOrder(Order $order): string
{
    $accessToken = self::getPaypalAccessToken();
    $baseUrl = config('payment.paypal.mode') === 'sandbox'
        ? 'https://api-m.sandbox.paypal.com'
        : 'https://api-m.paypal.com';

    $usdAmount = round($order->total / config('payment.paypal.usd_rate'), 2);

    $response = Http::withToken($accessToken)
        ->post("{$baseUrl}/v2/checkout/orders", [
            'intent' => 'CAPTURE',
            'purchase_units' => [
                [
                    'reference_id' => $order->order_number,
                    'amount' => [
                        'currency_code' => 'USD',
                        'value' => number_format($usdAmount, 2, '.', ''),
                    ],
                    'description' => "Order {$order->order_number}",
                ],
            ],
            'application_context' => [
                'return_url' => route('payment.paypal.return'),
                'cancel_url' => route('payment.paypal.cancel'),
                'brand_name' => config('app.name'),
                'user_action' => 'PAY_NOW',
            ],
        ]);

    $data = $response->json();

    // Store PayPal order ID in the order for reference
    $order->update([
        'payment_data' => ['paypal_order_id' => $data['id']],
    ]);

    // Find and return the approve URL
    $approveLink = collect($data['links'])
        ->firstWhere('rel', 'approve');

    return $approveLink['href'];
}
```

### USD conversion

The conversion formula: `$usdAmount = round($order->total / config('payment.paypal.usd_rate'), 2)`

Example: If order total is 500,000 VND and usd_rate is 25,000, the PayPal amount is $20.00 USD.

### CSRF exclusion

In `app/Http/Middleware/VerifyCsrfToken.php`:

```php
protected $except = [

    'payment/paypal/*',
];
```

Or use route-level `withoutMiddleware` if the project uses that pattern.
