# 33 — VNPay payment integration

**Size:** M (2h)

## Description

Implement the VNPay payment gateway integration: the return URL handler (browser redirect after payment), the IPN (Instant Payment Notification) callback handler, and CSRF exclusion for VNPay callback routes. This task creates the PaymentController with VNPay-specific methods and ensures the checkout flow correctly redirects to VNPay when that payment method is selected.

This task depends on task 30 (PaymentService must exist). It can be worked on in parallel with tasks 31, 34, 35, 36.

## Acceptance Criteria

1. When a user selects VNPay at checkout, they are redirected to the VNPay payment page.
2. After payment, VNPay redirects the user back to the return URL, where the hash is verified.
3. On valid payment: order payment_status is updated to "paid", user sees success page.
4. On failed/cancelled payment: order payment_status is updated to "failed", user sees error message.
5. VNPay IPN callback updates order status and returns correct JSON response `{"RspCode":"00"}`.
6. IPN endpoint is excluded from CSRF verification.
7. Payment data (VNPay response) is stored in the order's `payment_data` JSON column.
8. Works correctly with VNPay sandbox credentials.

## Technical Details

### `app/Http/Controllers/PaymentController.php`

```php
class PaymentController extends Controller
{
    /**
     * GET /payment/vnpay/return
     * Handles the browser redirect from VNPay after payment.
     */
    public function vnpayReturn(Request $request): RedirectResponse
    {
        $params = $request->all();

        if (!PaymentService::verifyVnpayHash($params)) {
            return redirect()->route('home')
                ->with('error', __('Payment verification failed'));
        }

        $order = Order::where('order_number', $params['vnp_TxnRef'])->firstOrFail();

        if ($params['vnp_ResponseCode'] === '00') {
            // Payment successful
            $order->update([
                'payment_status' => 'paid',
                'payment_data'   => $params,
            ]);
            return redirect()->route('orders.success', $order);
        } else {
            // Payment failed or cancelled
            $order->update([
                'payment_status' => 'failed',
                'payment_data'   => $params,
            ]);
            return redirect()->route('cart.index')
                ->with('error', __('Payment failed. Please try again.'));
        }
    }

    /**
     * GET /payment/vnpay/ipn
     * Handles VNPay server-to-server IPN notification.
     * Returns JSON response — VNPay expects {"RspCode":"00","Message":"Confirm Success"}.
     */
    public function vnpayIpn(Request $request): JsonResponse
    {
        $params = $request->all();

        // Verify hash
        if (!PaymentService::verifyVnpayHash($params)) {
            return response()->json([
                'RspCode' => '97',
                'Message' => 'Invalid signature',
            ]);
        }

        $order = Order::where('order_number', $params['vnp_TxnRef'])->first();

        if (!$order) {
            return response()->json([
                'RspCode' => '01',
                'Message' => 'Order not found',
            ]);
        }

        // Check if order already processed
        if ($order->payment_status === 'paid') {
            return response()->json([
                'RspCode' => '02',
                'Message' => 'Order already confirmed',
            ]);
        }

        if ($params['vnp_ResponseCode'] === '00') {
            $order->update([
                'payment_status' => 'paid',
                'payment_data'   => $params,
            ]);
        } else {
            $order->update([
                'payment_status' => 'failed',
                'payment_data'   => $params,
            ]);
        }

        return response()->json([
            'RspCode' => '00',
            'Message' => 'Confirm Success',
        ]);
    }
}
```

### CSRF exclusion

Add VNPay routes to CSRF exclusion. Two approaches (use whichever matches the project):

**Option A: VerifyCsrfToken middleware `$except` array**

In `app/Http/Middleware/VerifyCsrfToken.php`:

```php
protected $except = [
    'payment/vnpay/*',
];
```

**Option B: Route-level exclusion (Laravel 11+)**

In route definition:
```php
Route::get('/payment/vnpay/ipn', [PaymentController::class, 'vnpayIpn'])
    ->withoutMiddleware([\App\Http\Middleware\VerifyCsrfToken::class]);
```
