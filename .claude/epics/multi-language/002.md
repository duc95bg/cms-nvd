# Task 002 — Middleware SetLocale

## Mục tiêu
Tạo middleware đọc locale từ URL segment hoặc session và áp dụng vào request.

## Việc cần làm
1. Tạo `app/Http/Middleware/SetLocale.php`:
   - Constant `SUPPORTED = ['en', 'vi']`, `DEFAULT = 'vi'`.
   - Lấy `$request->segment(1)`.
   - Nếu thuộc SUPPORTED → set locale + `session()->put('locale', ...)`.
   - Ngược lại → đọc `session('locale', DEFAULT)`.
   - Gọi `app()->setLocale($locale)`.
