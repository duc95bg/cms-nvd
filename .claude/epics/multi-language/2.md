# Task 2 — Lang files & config

## Mục tiêu
Chuẩn bị file dịch JSON và cấu hình locale mặc định.

## Việc cần làm
1. Tạo `lang/en.json` với các key: `welcome`, `home`, `about`, `login`, `logout`, `language`, `hello_name`.
2. Tạo `lang/vi.json` với bản dịch tương ứng.
3. Sửa `config/app.php`:
   - `'locale' => 'vi'`
   - `'fallback_locale' => 'en'`

## Acceptance criteria
- `php artisan tinker` → `__('welcome')` trả về "Chào mừng đến với CMS" khi `app()->setLocale('vi')`.
- `__('welcome')` trả về "Welcome to CMS" khi setLocale('en').
- `php artisan config:cache` chạy không lỗi.

## Files đụng tới
- `lang/en.json` (new)
- `lang/vi.json` (new)
- `config/app.php`
