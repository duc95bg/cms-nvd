# Task 4 — Đăng ký middleware & routes

## Mục tiêu
Gắn `SetLocale` vào pipeline web và thêm route group minh hoạ.

## Việc cần làm
1. Sửa `bootstrap/app.php`:
   ```php
   ->withMiddleware(function (Middleware $middleware): void {
       $middleware->web(append: [\App\Http\Middleware\SetLocale::class]);
       $middleware->alias(['locale' => \App\Http\Middleware\SetLocale::class]);
   })
   ```
2. Sửa `routes/web.php`:
   - Root `/` redirect tới `/'.app()->getLocale()`.
   - Group `prefix({locale})->where(['locale' => 'en|vi'])` với 2 route demo: `home`, `about`.

## Acceptance criteria
- `php artisan route:list` hiển thị `GET /{locale}`, `GET /{locale}/about`.
- `GET /` redirect về `/vi` khi chưa có session.
- Không lỗi `Target class [locale] does not exist` khi gọi middleware.

## Files đụng tới
- `bootstrap/app.php`
- `routes/web.php`
