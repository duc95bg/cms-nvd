# Task 6 — Feature tests

## Mục tiêu
Bảo đảm 4 kịch bản locale hoạt động đúng.

## Việc cần làm
Tạo `tests/Feature/MultiLanguageTest.php` với 4 test:
1. `GET /en` → status 200 và response chứa "Welcome to CMS".
2. `GET /vi` → response chứa "Chào mừng đến với CMS".
3. Visit `/en` trước, sau đó `GET /about` (no prefix, cùng session) → tiếng Anh.
4. Clean session, `GET /dashboard` không tồn tại nhưng middleware vẫn set locale `vi` (dùng `app()->getLocale()` assertion qua một route test riêng hoặc inspect session).

## Acceptance criteria
- `php artisan test --filter=MultiLanguageTest` pass 4/4.
- Không có warning deprecation.
